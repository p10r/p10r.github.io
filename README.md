# Brutal - The neobrutalist Astro theme

Brutal is a minimal neobrutalist theme for [Astro](https://astro.build/). It's based on Neobrutalist Web Design, a
movement that aims to create websites with a minimalistic and functional design. It has some integrations like Image
Optimization, RSS, Sitemap, ready to get your SEO done right.

## Integrations

### RSS

The RSS feed is automatically generated from the Markdown files in the `src/content/blog` folder. You can ofcourse
completely change this to your own needs.

The RSS will output to `https://example.com/feed.xml` by default. You can change this, by renaming
`src/pages/feed.xml.js`.
